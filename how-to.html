How To Make A Hardware Page

1. The first step I took was making a new HTML file in my cs-portfolio folder. I named this file "hardware.html."
2. Next, I created my navigation bar. To do this, I constructed the links for my web pages and put those into an unordered list. The tag for an unodered list is ul and the one for a listed item is li.
3. I added my style tag next, to add borders to my text. The "p" tag is to add a paragraph. Text align is to align my text and font size, width, padding, border, and margin all controlled how my border would end up looking.
4. To add images, I added the image tag, which is "img." Also, I added the source using my image link.
5. Finally, I applied my "div" tag. This provided the text inside my borders. It gave clear descriptions on the computer components.
